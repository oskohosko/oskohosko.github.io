I'll be using this site as a canvas to showcase my exploration and to experiment with new things I'm learning.

A lot of my projects will be focused on the application to my hobbies. I'm very interested in how I can combine the tech side of my life with my hobbies to improve and discover more about them.

For example, here are some things I'd like to explore:

  • How can I improve my golf game using Python?

  • Common patterns/habits in my Chess games

  • Using AI and APIs to generate cover images for my Spotify playlists


On this page, I'll be posting my most recent project(s) below and any other things I think are worthwhile showing here!


Spotify Playlist Analyser and Cover Image Generator

Using Python, Spotify, Musixmatch and OpenAI's APIs, I've made a program that takes a spotify playlist, grabs track data, lyrics from Musixmatch and uses GPT 3.5 and DALLE 2 to analyse the lyrics and generate cover images for the playlist.
