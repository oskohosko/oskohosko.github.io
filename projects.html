House Preferences Web App

This is a web application using Python, Flask, SQLite/Postgres and HTML. Each year my mates and I go for a trip and we sort ourselves into 3 houses of roughly 8 people. This is a web app dedicated to this trip, which allows us to sign up, log in and cast two votes for who we want to be in a house with for the trip.

I have deployed this app using Heroku and on the published version I used a Postgres Database rather than SQLite.




Spotify Playlist Cover Image Maker with Analysis

Using Python, Spotify, Musixmatch and OpenAI's APIs, I've made a program that takes a spotify playlist, grabs track data, lyrics from Musixmatch and uses GPT 3.5 and DALL-E 2 to analyse the lyrics and generate cover images for the playlist based on the analysis.

Need for Speed Heat: Optimal Path/Race Finding


One of the video games I've been playing a bit recently is Need for Speed: Heat. And one way of making money in the game is by racing. I'm currently doing an AI unit this semester and am learning about search algorithms and I thought it would be cool to apply this to Need for Speed. So, in this project I am trying to use search algorithms to find the best sequence of races I can take in a given amount of time that will give me the most money.


Spotify Recommended Track Navigator


Following on from my other Spotify focused project, in this one I am going to choose a starting song and an ending song, and based purely on recommended songs, finding a sequence of songs that leads from the start song to the end song. For example, I could choose a happy song to start with and a sad song to end with and by gathering recommended songs for each song, finding a path of songs that leads from the happy one to the sad one to create a playlist.
