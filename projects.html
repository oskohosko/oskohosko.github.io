Spotify Playlist Analyser and Cover Image Generator

Using Python, Spotify, Musixmatch and OpenAI's APIs, I've made a program that takes a spotify playlist, grabs track data, lyrics from Musixmatch and uses GPT 3.5 and DALLE 2 to analyse the lyrics and generate cover images for the playlist.
